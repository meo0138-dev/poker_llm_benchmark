About the Benchmark

Methodology

Our benchmark evaluates LLMs on strategic poker reasoning through 26 carefully crafted scenarios. Each question tests different aspects of poker knowledge including hand evaluation, pot odds calculation, position play, and bluffing strategies. Each question has only 1 correct answer which is verified using the solver GTO Wizard. Models are evaluated on accuracy.

Evaluation Criteria

  • Correct action selection (fold/call/raise)
  • Accurate probability estimation
Note: The dataset is not publicly available to prevent contamination of future model training.
